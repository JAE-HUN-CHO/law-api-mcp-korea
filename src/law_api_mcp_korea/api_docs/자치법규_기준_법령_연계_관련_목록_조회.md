# 자치법규 기준 법령 연계 관련 목록 조회

## 1. API 명칭 및 기본 설명
- **API 명칭**: 자치법규 기준 법령 연계 관련 목록 조회
- **기본 설명**: 자치법규 기준 연계 조례 목록/연계 법령별 조례 목록/지자체별 목록을 함께 제공하는 묶음 가이드

## 2. 가이드 페이지 URL
- https://open.law.go.kr/LSO/openApi/guideResult.do?htmlName=ordinLsConListGuide

## 3. 요청 URL (Endpoint)
### 3-1. 연계 조례 목록 조회
- http://www.law.go.kr/DRF/lawSearch.do?target=lnkOrd

### 3-2. 연계 법령별 조례 목록 조회
- http://www.law.go.kr/DRF/lawSearch.do?target=lnkLsOrd

### 3-3. 연계 조례 지자체별 목록 조회
- http://www.law.go.kr/DRF/lawSearch.do?target=lnkOrg

## 4. 요청 변수 (Request Parameters) 명세표
### 4-1. 연계 조례 목록 조회
| 변수명 | 타입/필수 여부 | 설명 |
| --- | --- | --- |
| OC | string, 필수 | 사용자 이메일 ID |
| target | string : lnkOrd, 필수 | 서비스 대상 |
| type | char, 필수 | 출력 형태 : HTML/XML/JSON |
| ID | char | 자치법규 ID |
| LID | char | 자치법규 일련번호 |
| LM | string | 자치법규명 |
| display | int | 검색 개수 |
| page | int | 결과 페이지 |

### 4-2. 연계 법령별 조례 목록 조회
| 변수명 | 타입/필수 여부 | 설명 |
| --- | --- | --- |
| OC | string, 필수 | 사용자 이메일 ID |
| target | string : lnkLsOrd, 필수 | 서비스 대상 |
| type | char, 필수 | 출력 형태 : HTML/XML/JSON |
| ID | char | 법령 ID |
| display | int | 검색 개수 |
| page | int | 결과 페이지 |

### 4-3. 연계 조례 지자체별 목록 조회
| 변수명 | 타입/필수 여부 | 설명 |
| --- | --- | --- |
| OC | string, 필수 | 사용자 이메일 ID |
| target | string : lnkOrg, 필수 | 서비스 대상 |
| type | char, 필수 | 출력 형태 : HTML/XML/JSON |
| org | string | 지자체 코드 |
| display | int | 검색 개수 |
| page | int | 결과 페이지 |

## 6. 에러 코드
- 가이드 페이지에 별도 에러 코드 표는 명시되지 않은 경우가 많습니다. 일반적으로 HTTP 오류, 빈 결과, 또는 XML/JSON 응답의 상태값(제공 시)으로 실패를 판별합니다.

## 샘플 요청 및 응답 예시
- 아래 예시는 호출 형태를 빠르게 확인하기 위한 템플릿입니다.
- `{...}` 형태의 식별자 값은 실제 조회 대상 값으로 치환해서 사용합니다.

### 연계 조례 목록 조회
#### 요청 예시
- 예시 1 (XML)
```text
http://www.law.go.kr/DRF/lawSearch.do?target=lnkOrd&OC=test&type=XML&LM=자동차관리법&display=20&page=1
```
- 예시 2 (JSON)
```text
http://www.law.go.kr/DRF/lawSearch.do?target=lnkOrd&OC=test&type=JSON&LM=자동차관리법&display=20&page=1
```
#### 응답 예시
- XML 구조 예시
```xml
<response>
  <item>
    <field name="자치법규ID">12345</field>
    <field name="자치법규명">서울특별시 주차장 조례</field>
    <field name="연계법령목록">예시값</field>
    <field name="연계건수">20</field>
    <field name="상세링크">https://www.law.go.kr/...</field>
  </item>
</response>
```
- JSON 구조 예시
```json
{
  "data": [
    {
      "자치법규ID": 12345,
      "자치법규명": "서울특별시 주차장 조례",
      "연계법령목록": "예시값",
      "연계건수": 20,
      "상세링크": "https://www.law.go.kr/..."
    }
  ]
}
```
