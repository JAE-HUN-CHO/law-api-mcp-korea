# 산업재해보상보험재심사위원회 결정문 본문 조회

## 1. API 명칭 및 기본 설명
- **API 명칭**: 산업재해보상보험재심사위원회 결정문 본문 조회
- **기본 설명**: 산업재해보상보험재심사위원회 결정문 본문을 조회하는 API

## 2. 가이드 페이지 URL
- https://open.law.go.kr/LSO/openApi/guideResult.do?htmlName=iaciacInfoGuide

## 3. 요청 URL (Endpoint)
- https://www.law.go.kr/DRF/lawService.do?target=iaciac

## 4. 요청 변수 (Request Parameters) 명세표
| 변수명 | 타입/필수 여부 | 설명 |
| --- | --- | --- |
| OC | string(필수) | 신청한 API인증값 |
| target | string(필수) | 서비스 대상 (산업재해보상보험재심사위원회 : iaciac) |
| type | char(필수) | 출력 형태 : HTML/XML/JSON |
| ID | char(필수) | 결정문 일련번호 |

## 5. 출력 결과 (Response Elements) 명세표
| 필드명 | 타입 | 설명 |
| --- | --- | --- |
| 결정문 일련번호 | int | 결정문 일련번호 |
| 사건대분류 | string | 사건 대분류 |
| 사건중분류 | string | 사건 중분류 |
| 사건소분류 | string | 사건 소분류 |
| 쟁점 | string | 쟁점 |
| 사건번호 | string | 사건번호 |
| 의결일자 | string | 의결일자 |
| 사건 | string | 사건 |
| 청구인 | string | 청구인 |
| 재해근로자 | string | 재해근로자 |
| 재해자 | string | 재해자 |
| 피재근로자 | string | 피재근로자/피재자성명/피재자/피재자(망인) |
| 진폐근로자 | string | 진폐근로자 |
| 수진자 | string | 수진자 |
| 원처분기관 | string | 원처분기관 |
| 주문 | string | 주문 |
| 청구취지 | string | 청구취지 |
| 이유 | string | 이유 |
| 별지 | string | 별지 |
| 문서제공구분 | string | 문서제공구분(데이터 개방|이유하단 이미지개방) |
| 각주번호 | int | 각주번호 |
| 각주내용 | string | 각주내용 |

## 6. 에러 코드
- 가이드 페이지에 별도 에러 코드 표는 명시되지 않은 경우가 많습니다. 일반적으로 HTTP 오류, 빈 결과, 또는 XML/JSON 응답의 상태값(제공 시)으로 실패를 판별합니다.

## 7. 비고
- 노동위원회(nlrc) 대표 가이드 구조를 기준으로 동일 계열 위원회 API를 정규화했습니다.

## 샘플 요청 및 응답 예시
- 아래 예시는 호출 형태를 빠르게 확인하기 위한 템플릿입니다.
- `{...}` 형태의 식별자 값은 실제 조회 대상 값으로 치환해서 사용합니다.

#### 요청 예시
- 예시 1 (HTML)
```text
https://www.law.go.kr/DRF/lawService.do?OC=test&target=iaciac&ID=7515&type=HTML
```
- 예시 2 (XML)
```text
https://www.law.go.kr/DRF/lawService.do?OC=test&target=iaciac&ID=7513&type=XML
```
- 예시 3 (JSON)
```text
https://www.law.go.kr/DRF/lawService.do?OC=test&target=iaciac&ID=12713&type=JSON
```
