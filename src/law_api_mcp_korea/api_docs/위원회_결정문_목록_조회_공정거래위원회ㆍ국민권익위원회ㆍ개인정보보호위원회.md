# 위원회 결정문 목록 조회 (공정거래위원회·국민권익위원회·개인정보보호위원회)

## 1. API 명칭 및 기본 설명
- **API 명칭**: 위원회 결정문 목록 조회 (공정거래위원회·국민권익위원회·개인정보보호위원회)
- **기본 설명**: 공정거래위원회(ftc), 국민권익위원회(acr), 개인정보보호위원회(ppc)의 결정문 목록 조회를 한 페이지에 묶어 제공하는 가이드

## 2. 가이드 페이지 URL
- https://open.law.go.kr/LSO/openApi/guideResult.do?htmlName=cmtListGuide

## 3. 요청 URL (Endpoint)
### 3-1. 공정거래위원회(ftc) 결정문 목록
- https://www.law.go.kr/DRF/lawSearch.do?target=ftc

### 3-2. 국민권익위원회(acr) 결정문 목록
- https://www.law.go.kr/DRF/lawSearch.do?target=acr

### 3-3. 개인정보보호위원회(ppc) 결정문 목록
- https://www.law.go.kr/DRF/lawSearch.do?target=ppc

| 변수명 | 타입/필수 여부 | 설명 |
| --- | --- | --- |
| OC | string, 필수 | 사용자 이메일 ID |
| target | string : ftc(필수) | 서비스 대상(ftc/acr/ppc) |
| type | char, 필수 | 출력 형태 : HTML/XML/JSON |
| search | int | 검색범위(기본: 제목, 2 본문검색) |
| query | string | 검색어 |
| display | int | 검색 개수 |
| page | int | 결과 페이지 |
| sort | string | 정렬옵션 |
| gana | string | 사전식 검색 |

## 6. 에러 코드
- 가이드 페이지에 별도 에러 코드 표는 명시되지 않은 경우가 많습니다. 일반적으로 HTTP 오류, 빈 결과, 또는 XML/JSON 응답의 상태값(제공 시)으로 실패를 판별합니다.

## 7. 비고
- 공정거래위원회·국민권익위원회·개인정보보호위원회 3개 target을 하나의 가이드에서 함께 안내합니다.

## 샘플 요청 및 응답 예시
- 아래 예시는 호출 형태를 빠르게 확인하기 위한 템플릿입니다.

### 공정거래위원회(ftc) 결정문 목록
#### 요청 예시
- 예시 1 (XML)
```text
https://www.law.go.kr/DRF/lawSearch.do?target=ftc&OC=test&type=XML&search=1&query=사건&display=20&page=1
```
- 예시 2 (HTML)
```text
https://www.law.go.kr/DRF/lawSearch.do?target=ftc&OC=test&type=HTML&search=1&query=사건&display=20&page=1
```
#### 응답 예시
- XML 구조 예시
```xml
<response>
  <item>
    <field name="target">예시값</field>
    <field name="키워드">사건</field>
    <field name="section">예시값</field>
    <field name="totalCnt">20</field>
    <field name="제목">예시 사건</field>
    <field name="상세링크">https://www.law.go.kr/...</field>
  </item>
</response>
```
