# 조세심판원 특별행정심판례 목록 조회

## 1. API 명칭 및 기본 설명
- **API 명칭**: 조세심판원 특별행정심판례 목록 조회
- **기본 설명**: 조세심판원 특별행정심판재결례 목록 조회 API

## 2. 가이드 페이지 URL
- https://open.law.go.kr/LSO/openApi/guideResult.do?htmlName=specialDeccTtListGuide

## 3. 요청 URL (Endpoint)
- http://www.law.go.kr/DRF/lawSearch.do?target=ttSpecialDecc

## 4. 요청 변수 (Request Parameters) 명세표
| 변수명 | 타입/필수 여부 | 설명 |
| --- | --- | --- |
| OC | string, 필수 | 사용자 이메일 ID |
| target | string : ttSpecialDecc(필수) | 서비스 대상(기관별 special target) |
| type | char, 필수 | 출력 형태 : HTML/XML/JSON |
| search | int | 검색범위(기본: 1 재결례명, 2 본문검색) |
| query | string | 검색어 |
| display | int | 검색된 결과 개수(default=20, max=100) |
| page | int | 검색 결과 페이지(default=1) |
| cls | string | 재결례 유형 |
| gana | string | 사전식 검색(ga,na,da…) |
| date | int | 의결일자 |
| dpaYd | string | 처분일자 기간검색 |
| rslYd | string | 의결일자 기간검색 |
| sort | string | 정렬옵션(lasc/ldes/dasc/ddes/nasc/ndes) |
| popYn | string | 상세화면 팝업 여부 |
| fields | string | 응답항목 옵션(HTML 형식에서는 미적용) |

## 5. 출력 결과 (Response Elements) 명세표
| 필드명 | 타입 | 설명 |
| --- | --- | --- |
| target | string | 검색 대상 |
| 키워드 | string | 키워드 |
| section | string | 검색범위(EvtNm:재결례명 / bdyText:본문) |
| totalCnt | int | 검색결과갯수 |
| page | int | 출력페이지 |
| id | int | 검색결과번호 |
| 특별행정심판재결례일련번호 | int | 특별행정심판재결례일련번호 |
| 사건명 | string | 사건명 |
| 청구번호 | string | 청구번호 |
| 처분일자 | string | 처분일자 |
| 의결일자 | string | 의결일자 |
| 처분청 | string | 처분청 |
| 재결청 | int | 재결청 |
| 재결구분명 | string | 재결구분명 |
| 재결구분코드 | string | 재결구분코드 |
| 데이터기준일시 | string | 데이터기준일시 |
| 행정심판재결례상세링크 | string | 상세링크 |

## 6. 에러 코드
- 가이드 페이지에 별도 에러 코드 표는 명시되지 않은 경우가 많습니다. 일반적으로 HTTP 오류, 빈 결과, 또는 XML/JSON 응답의 상태값(제공 시)으로 실패를 판별합니다.

## 7. 비고
- 조세심판원 대표 가이드 구조를 기준으로 특별행정심판 계열 API를 정규화했습니다.

## 샘플 요청 및 응답 예시
- 아래 예시는 호출 형태를 빠르게 확인하기 위한 템플릿입니다.

#### 요청 예시
- 예시 1 (XML)
```text
http://www.law.go.kr/DRF/lawSearch.do?target=ttSpecialDecc&OC=test&type=XML&search=1&query=자동차&display=20&page=1
```
- 예시 2 (JSON)
```text
http://www.law.go.kr/DRF/lawSearch.do?target=ttSpecialDecc&OC=test&type=JSON&search=1&query=자동차&display=20&page=1
```
#### 응답 예시
- XML 구조 예시
```xml
<response>
  <item>
    <field name="target">예시값</field>
    <field name="키워드">자동차</field>
    <field name="section">예시값</field>
    <field name="totalCnt">20</field>
    <field name="사건명">예시 사건</field>
    <field name="행정심판재결례상세링크">https://www.law.go.kr/...</field>
  </item>
</response>
```
- JSON 구조 예시
```json
{
  "data": [
    {
      "target": "예시값",
      "키워드": "자동차",
      "section": "예시값",
      "totalCnt": 20,
      "사건명": "예시 사건",
      "행정심판재결례상세링크": "https://www.law.go.kr/..."
    }
  ]
}
```
