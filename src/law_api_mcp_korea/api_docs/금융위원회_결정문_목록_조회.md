# 금융위원회 결정문 목록 조회

## 1. API 명칭 및 기본 설명
- **API 명칭**: 금융위원회 결정문 목록 조회
- **기본 설명**: 금융위원회 결정문 목록을 조회하는 API

## 2. 가이드 페이지 URL
- https://open.law.go.kr/LSO/openApi/guideResult.do?htmlName=fscListGuide

## 3. 요청 URL (Endpoint)
- https://www.law.go.kr/DRF/lawSearch.do?target=fsc

## 4. 요청 변수 (Request Parameters) 명세표
| 변수명 | 타입/필수 여부 | 설명 |
| --- | --- | --- |
| OC | string, 필수 | 사용자 이메일 ID |
| target | string(필수) (금융위원회 : fsc) | 서비스 대상(위원회별 target 코드) |
| type | char, 필수 | 출력 형태 : HTML/XML/JSON |
| search | int | 검색범위(기본: 1 제목, 2 본문검색) |
| query | string | 검색어 |
| display | int | 검색된 결과 개수(default=20, max=100) |
| page | int | 검색 결과 페이지(default=1) |
| gana | string | 사전식 검색(ga,na,da…) |
| sort | string | 정렬옵션(lasc/ldes/dasc/ddes/nasc/ndes 등) |
| popYn | string | 상세화면 팝업 여부 |

## 5. 출력 결과 (Response Elements) 명세표
| 필드명 | 타입 | 설명 |
| --- | --- | --- |
| target | string | 검색서비스 대상 |
| 키워드 | string | 검색 단어 |
| section | string | 검색범위 |
| totalCnt | int | 검색 건수 |
| page | int | 현재 페이지번호 |
| 기관명 | string | 위원회명 |
| id | int | 검색 결과 순번 |
| 결정문일련번호 | int | 결정문 일련번호 |
| 제목 | string | 제목 |
| 사건번호 | string | 사건번호 |
| 등록일 | string | 등록일 |
| 결정문상세링크 | string | 결정문 상세링크 |

## 6. 에러 코드
- 가이드 페이지에 별도 에러 코드 표는 명시되지 않은 경우가 많습니다. 일반적으로 HTTP 오류, 빈 결과, 또는 XML/JSON 응답의 상태값(제공 시)으로 실패를 판별합니다.

## 7. 비고
- 노동위원회(nlrc) 대표 가이드 구조를 기준으로 동일 계열 위원회 API를 정규화했습니다.

## 샘플 요청 및 응답 예시
- 아래 예시는 호출 형태를 빠르게 확인하기 위한 템플릿입니다.

#### 요청 예시
- 예시 1 (XML)
```text
https://www.law.go.kr/DRF/lawSearch.do?target=fsc&OC=test&type=XML&search=1&query=사건&display=20&page=1
```
- 예시 2 (JSON)
```text
https://www.law.go.kr/DRF/lawSearch.do?target=fsc&OC=test&type=JSON&search=1&query=사건&display=20&page=1
```
