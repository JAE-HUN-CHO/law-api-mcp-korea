# 현행법령(공포일) 본문 조회

## 1. API 명칭 및 기본 설명
- **API 명칭**: 현행법령(공포일) 본문 조회
- **기본 설명**: 현행법령(공포일 기준)의 본문을 조회하는 상세 API

## 2. 가이드 페이지 URL
- https://open.law.go.kr/LSO/openApi/guideResult.do?htmlName=lsNwInfoGuide

## 3. 요청 URL (Endpoint)
- http://www.law.go.kr/DRF/lawService.do?target=law

## 4. 요청 변수 (Request Parameters) 명세표
| 변수명 | 타입/필수 여부 | 설명 |
| --- | --- | --- |
| OC | string(필수) | 신청한 API인증값 |
| target | string : law(필수) | 서비스 대상 |
| type | char(필수) | 출력 형태 : HTML/XML/JSON |
| ID | char | 법령 ID (ID 또는 MST 중 하나는 반드시 입력) |
| MST | char | 법령 마스터 번호 - 법령테이블의 lsi_seq 값을 의미함 |
| LM | string | 법령의 법령명(법령명 입력시 해당 법령 링크) |
| LD | int | 법령의 공포일자 |
| LN | int | 법령의 공포번호 |
| JO | int | 조번호 생략(기본값) : 모든 조를 표시함 6자리숫자 : 조번호(4자리)+조가지번호(2자리) (000200 : 2조, 001002 : 10조의 2) |
| LANG | char | 원문/한글 여부 생략(기본값) : 한글 (KO : 한글, ORI : 원문) |

## 6. 에러 코드
- 가이드 페이지에 별도 에러 코드 표는 명시되지 않은 경우가 많습니다. 일반적으로 HTTP 오류, 빈 결과, 또는 XML/JSON 응답의 상태값(제공 시)으로 실패를 판별합니다.

## 샘플 요청 및 응답 예시
- 아래 예시는 호출 형태를 빠르게 확인하기 위한 템플릿입니다.
- `{...}` 형태의 식별자 값은 실제 조회 대상 값으로 치환해서 사용합니다.

#### 요청 예시
- 예시 1 (HTML)
```text
http://www.law.go.kr/DRF/lawService.do?OC=test&target=law&ID=009682&type=HTML
```
- 예시 2 (HTML)
```text
http://www.law.go.kr/DRF/lawService.do?OC=test&target=law&MST=261457&type=HTML
```
- 예시 3 (XML)
```text
http://www.law.go.kr/DRF/lawService.do?OC=test&target=law&ID=009682&type=XML
```
- 예시 4 (XML)
```text
http://www.law.go.kr/DRF/lawService.do?OC=test&target=law&MST=261457&type=XML
```
- 예시 5 (JSON)
```text
http://www.law.go.kr/DRF/lawService.do?OC=test&target=law&ID=009682&type=JSON
```
- 예시 6 (JSON)
```text
http://www.law.go.kr/DRF/lawService.do?OC=test&target=law&MST=261457&type=JSON
```

#### 응답 예시
- XML 구조 예시
```xml
<response>
  <item>
    <field name="기본정보">예시값</field>
    <field name="법령ID">12345</field>
    <field name="법령일련번호">12345</field>
    <field name="법령명">자동차관리법</field>
    <field name="조문내용">예시 본문</field>
    <field name="상세링크/첨부">https://www.law.go.kr/...</field>
  </item>
</response>
```
- JSON 구조 예시
```json

  "data": [

      "기본정보": "예시값",

```
