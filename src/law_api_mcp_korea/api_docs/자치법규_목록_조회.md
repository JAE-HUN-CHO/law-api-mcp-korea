# 자치법규 목록 조회

## 1. API 명칭 및 기본 설명
- **API 명칭**: 자치법규 목록 조회
- **기본 설명**: 자치법규 목록을 조회하는 API

## 2. 가이드 페이지 URL
- https://open.law.go.kr/LSO/openApi/guideResult.do?htmlName=ordinListGuide

## 3. 요청 URL (Endpoint)
- http://www.law.go.kr/DRF/lawSearch.do?target=ordin

## 4. 요청 변수 (Request Parameters) 명세표
| 변수명 | 타입/필수 여부 | 설명 |
| --- | --- | --- |
| OC | string, 필수 | 사용자 이메일 ID |
| target | string : ordin, 필수 | 서비스 대상 |
| type | char, 필수 | 출력 형태 : HTML/XML/JSON |
| search | int | 검색범위(기본: 1 자치법규명, 2 본문검색) |
| query | string | 검색어 |
| display | int | 검색된 결과 개수(default=20, max=100) |
| page | int | 검색 결과 페이지(default=1) |
| org | string | 지자체 코드(시·도) |
| sborg | string | 지자체 코드(시·군·구) |
| gana | string | 사전식 검색 |
| sort | string | 정렬옵션(lasc/ldes/dasc/ddes/nasc/ndes) |
| date | int | 공포일자 |
| prmlYd | string | 공포일자 기간검색 |
| popYn | string | 상세화면 팝업 여부 |

## 5. 출력 결과 (Response Elements) 명세표
| 필드명 | 타입 | 설명 |
| --- | --- | --- |
| target | string | 검색서비스 대상 |
| 키워드 | string | 검색어 |
| section | string | 검색범위 |
| totalCnt | int | 검색건수 |
| page | int | 결과페이지번호 |
| ordin id | int | 결과 번호 |
| 자치법규일련번호 | int | 자치법규일련번호 |
| 자치법규명 | string | 자치법규명 |
| 자치단체명 | string | 자치단체명 |
| 공포일자 | int | 공포일자 |
| 공포번호 | string | 공포번호 |
| 제개정구분명 | string | 제개정구분명 |
| 자치법규ID | int | 자치법규 ID |
| 자치법규상세링크 | string | 자치법규상세링크 |

## 6. 에러 코드
- 가이드 페이지에 별도 에러 코드 표는 명시되지 않은 경우가 많습니다. 일반적으로 HTTP 오류, 빈 결과, 또는 XML/JSON 응답의 상태값(제공 시)으로 실패를 판별합니다.

## 샘플 요청 및 응답 예시
- 아래 예시는 호출 형태를 빠르게 확인하기 위한 템플릿입니다.

#### 요청 예시
- 예시 1 (XML)
```text
http://www.law.go.kr/DRF/lawSearch.do?target=ordin&OC=test&type=XML&search=1&query=조례&display=20&page=1
```
- 예시 2 (JSON)
```text
http://www.law.go.kr/DRF/lawSearch.do?target=ordin&OC=test&type=JSON&search=1&query=조례&display=20&page=1
```
#### 응답 예시
- XML 구조 예시
```xml
<response>
  <item>
    <field name="target">예시값</field>
    <field name="키워드">조례</field>
    <field name="section">예시값</field>
    <field name="totalCnt">20</field>
    <field name="자치법규상세링크">https://www.law.go.kr/...</field>
    <field name="page">1</field>
  </item>
</response>
```
- JSON 구조 예시
```json
{
  "data": [
    {
      "target": "예시값",
      "키워드": "조례",
      "section": "예시값",
      "totalCnt": 20,
      "자치법규상세링크": "https://www.law.go.kr/...",
      "page": 1
    }
  ]
}
```
