# 맞춤형 법령 조문 목록 조회

## 1. API 명칭 및 기본 설명
- **API 명칭**: 맞춤형 법령 조문 목록 조회
- **기본 설명**: 맞춤형 분류코드(vcode)에 따른 법령 조문 목록을 조회하는 API

## 2. 가이드 페이지 URL
- https://open.law.go.kr/LSO/openApi/guideResult.do?htmlName=custLsJoListGuide

## 3. 요청 URL (Endpoint)
- http://www.law.go.kr/DRF/lawSearch.do?target=couseLs&lj=jo

## 4. 요청 변수 (Request Parameters) 명세표
| 변수명 | 타입/필수 여부 | 설명 |
| --- | --- | --- |
| OC | string, 필수 | 사용자 이메일 ID |
| target | string : couseLs(필수) | 서비스 대상(couseLs/couseAdmrul/couseOrdin) |
| type | char, 필수 | 출력 형태 : HTML/XML/JSON |
| vcode | string, 필수 | 분류코드 |
| display | int | 검색된 결과 개수(default=20, max=100) |
| page | int | 검색 결과 페이지(default=1) |
| popYn | string | 상세화면 팝업 여부 |
| lj | string | jo 입력 시 조문 목록 조회 |

| 필드명 | 타입 | 설명 |
| --- | --- | --- |

| 법령ID | int | 법령ID |
| 공포일자 | int | 공포일자 |
| 공포번호 | int | 공포번호 |
| 제개정구분명 | string | 제개정구분명 |
| 소관부처명 | string | 소관부처명 |
| 소관부처코드 | string | 소관부처코드 |
| 법령구분명 | string | 법령구분명 |

| 조문제목 | string | 조문제목 |
| 조문내용 | string | 조문내용 |

## 6. 에러 코드
- 가이드 페이지에 별도 에러 코드 표는 명시되지 않은 경우가 많습니다. 일반적으로 HTTP 오류, 빈 결과, 또는 XML/JSON 응답의 상태값(제공 시)으로 실패를 판별합니다.

## 7. 비고
- lj=jo 옵션을 추가하여 조문 목록까지 확장 조회합니다.

## 샘플 요청 및 응답 예시
- 아래 예시는 호출 형태를 빠르게 확인하기 위한 템플릿입니다.
- `{...}` 형태의 식별자 값은 실제 조회 대상 값으로 치환해서 사용합니다.

#### 요청 예시
- 예시 1 (XML)
```text
http://www.law.go.kr/DRF/lawSearch.do?target=couseLs&lj=jo&OC=test&type=XML&vcode={분류코드}&display=20&page=1&lj=jo
```
- 예시 2 (JSON)
```text
http://www.law.go.kr/DRF/lawSearch.do?target=couseLs&lj=jo&OC=test&type=JSON&vcode={분류코드}&display=20&page=1&lj=jo
```
