# 조약 목록 조회

## 1. API 명칭 및 기본 설명
- **API 명칭**: 조약 목록 조회
- **기본 설명**: 조약 목록을 조회하는 API

## 2. 가이드 페이지 URL
- https://open.law.go.kr/LSO/openApi/guideResult.do?htmlName=trtyListGuide

## 3. 요청 URL (Endpoint)
- http://www.law.go.kr/DRF/lawSearch.do?target=trty

## 4. 요청 변수 (Request Parameters) 명세표
| 변수명 | 타입/필수 여부 | 설명 |
| --- | --- | --- |
| OC | string(필수) | 신청한 API인증값 |
| target | string : trty(필수) | 서비스 대상 |
| type | char(필수) | 출력 형태 : HTML/XML/JSON |
| search | int | 검색범위 (기본 : 1 조약명) 2 : 조약본문 |
| query | string | 검색범위에서 검색을 원하는 질의 (검색 결과 리스트) |
| display | int | 검색된 결과 개수 (default=20 max=100) |
| page | int | 검색 결과 페이지 (default=1) |
| gana | string | 사전식 검색 (ga,na,da…,etc) |
| eftYd | string | 발효일자 검색(20090101~20090130) |
| concYd | string | 체결일자 검색(20090101~20090130) |
| cls | int | 1 : 양자조약 2 : 다자조약 |
| natCd | int | 국가코드 |
| sort | string | 정렬옵션 (기본 : lasc 조약명오름차순) ldes 조약명내림차순 dasc : 발효일자 오름차순 ddes : 발효일자 내림차순 nasc : 조약번호 오름차순 ndes : 조약번호 내림차순 rasc : 관보게재일 오름차순 rdes : 관보게재일 내림차순 |
| popYn | string | 상세화면 팝업창 여부(팝업창으로 띄우고 싶을 때만 'popYn=Y') |
| ID | char | 조약 ID |

## 5. 출력 결과 (Response Elements) 명세표
| 필드명 | 타입 | 설명 |
| --- | --- | --- |
| target | string | 검색 대상 |
| 키워드 | string | 키워드 |
| section | string | 검색범위(TrtyNm:조약명/bdyText:본문) |
| totalCnt | int | 검색결과갯수 |
| page | int | 출력페이지 |
| trty id | int | 검색결과번호 |
| 조약일련번호 | int | 조약일련번호 |
| 조약명 | string | 조약명 |
| 조약구분코드 | string | 조약구분코드 |
| 조약구분명 | string | 조약구분명 |
| 발효일자 | string | 발효일자 |
| 서명일자 | string | 서명일자 |
| 관보게제일자 | string | 관보게제일자 |
| 조약번호 | int | 조약번호 |
| 국가번호 | int | 국가번호 |
| 조약상세링크 | string | 조약상세링크 |

## 6. 에러 코드
- 가이드 페이지에 별도 에러 코드 표는 명시되지 않은 경우가 많습니다. 일반적으로 HTTP 오류, 빈 결과, 또는 XML/JSON 응답의 상태값(제공 시)으로 실패를 판별합니다.

## 샘플 요청 및 응답 예시
- 아래 예시는 호출 형태를 빠르게 확인하기 위한 템플릿입니다.

#### 요청 예시
- 예시 1 (XML)
```text
http://www.law.go.kr/DRF/lawSearch.do?OC=test&target=trty&type=XML
```
- 예시 2 (HTML)
```text
http://www.law.go.kr/DRF/lawSearch.do?OC=test&target=trty&ID=284&type=HTML
```
- 예시 3 (JSON)
```text
http://www.law.go.kr/DRF/lawSearch.do?OC=test&target=trty&ID=284&type=JSON
```
- 예시 4 (XML)
```text
http://www.law.go.kr/DRF/lawSearch.do?OC=test&target=trty&type=XML&cls=2
```

#### 응답 예시
- XML 구조 예시
```xml
<response>
  <item>
    <field name="target">예시값</field>
    <field name="키워드">협정</field>
    <field name="section">예시값</field>
    <field name="totalCnt">20</field>
    <field name="조약상세링크">https://www.law.go.kr/...</field>
    <field name="page">1</field>
  </item>
</response>
```
- JSON 구조 예시
```json
{
  "data": [
    {
      "target": "예시값",
      "키워드": "협정",
      "section": "예시값",
      "totalCnt": 20,
      "조약상세링크": "https://www.law.go.kr/...",
      "page": 1
    }
  ]
}
```
