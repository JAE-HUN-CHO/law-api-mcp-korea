# 행정규칙 별표ㆍ서식 목록 조회

## 1. API 명칭 및 기본 설명
- **API 명칭**: 행정규칙 별표ㆍ서식 목록 조회
- **기본 설명**: 행정규칙 별표·서식 목록을 조회하는 API

## 2. 가이드 페이지 URL
- https://open.law.go.kr/LSO/openApi/guideResult.do?htmlName=admrulBylListGuide

## 3. 요청 URL (Endpoint)
- http://www.law.go.kr/DRF/lawSearch.do?target=admbyl

## 4. 요청 변수 (Request Parameters) 명세표
| 변수명 | 타입/필수 여부 | 설명 |
| --- | --- | --- |
| OC | string, 필수 | 사용자 이메일 ID |
| target | string : admbyl(필수) | 서비스 대상(licbyl/admbyl/ordinbyl) |
| type | char | 출력 형태 : HTML/XML/JSON |
| search | int | 검색범위(기본: 1 별표서식명, 2 소속 법령명/행정규칙명/자치법규명, 3 별표본문검색) |
| query | string | 검색어(default=*) |
| display | int | 검색된 결과 개수(default=20, max=100) |
| page | int | 검색 결과 페이지(default=1) |
| sort | string | 정렬옵션(lasc/ldes) |
| org | string | 소관부처/지자체 검색 |
| sborg | string | 자치법규별 하위 지자체 검색(ordinbyl 전용) |
| knd | string | 별표종류(별표/서식/별지/별도/부록 등) |
| gana | string | 사전식 검색 |
| popYn | string | 상세화면 팝업 여부 |

## 5. 출력 결과 (Response Elements) 명세표
| 필드명 | 타입 | 설명 |
| --- | --- | --- |
| target | string | 검색 대상 |
| 키워드 | string | 검색어 |
| section | string | 검색범위 |
| totalCnt | int | 검색건수 |
| page | int | 현재 페이지 |
| id | int | 결과 번호 |
| 별표서식일련번호 | int | 별표/서식 일련번호 |
| 별표서식명 | string | 별표/서식명 |
| 소속법령명 | string | 소속 법령/행정규칙/자치법규명 |
| 별표종류 | string | 별표종류 |
| 상세링크 | string | 상세링크 |

## 6. 에러 코드
- 가이드 페이지에 별도 에러 코드 표는 명시되지 않은 경우가 많습니다. 일반적으로 HTTP 오류, 빈 결과, 또는 XML/JSON 응답의 상태값(제공 시)으로 실패를 판별합니다.

## 샘플 요청 및 응답 예시
- 아래 예시는 호출 형태를 빠르게 확인하기 위한 템플릿입니다.

#### 요청 예시
- 예시 1 (XML)
```text
http://www.law.go.kr/DRF/lawSearch.do?target=admbyl&OC=test&type=XML&search=1&query=학교&display=20&page=1
```
- 예시 2 (JSON)
```text
http://www.law.go.kr/DRF/lawSearch.do?target=admbyl&OC=test&type=JSON&search=1&query=학교&display=20&page=1
```
#### 응답 예시
- XML 구조 예시
```xml
<response>
  <item>
    <field name="target">예시값</field>
    <field name="키워드">학교</field>

    <field name="totalCnt">20</field>
    <field name="소속법령명">자동차관리법</field>
    <field name="상세링크">https://www.law.go.kr/...</field>
  </item>
</response>
```
