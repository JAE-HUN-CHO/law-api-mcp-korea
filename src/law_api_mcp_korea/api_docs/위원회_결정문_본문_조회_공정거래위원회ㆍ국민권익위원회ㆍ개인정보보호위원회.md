# 위원회 결정문 본문 조회 (공정거래위원회·국민권익위원회·개인정보보호위원회)

## 1. API 명칭 및 기본 설명
- **API 명칭**: 위원회 결정문 본문 조회 (공정거래위원회·국민권익위원회·개인정보보호위원회)
- **기본 설명**: 공정거래위원회(ftc), 국민권익위원회(acr), 개인정보보호위원회(ppc)의 결정문 본문 조회를 한 페이지에 묶어 제공하는 가이드

## 2. 가이드 페이지 URL
- https://open.law.go.kr/LSO/openApi/guideResult.do?htmlName=ftcInfoGuide

## 3. 요청 URL (Endpoint)
- https://www.law.go.kr/DRF/lawService.do?target=ftc

## 4. 요청 변수 (Request Parameters) 명세표
| 변수명 | 타입/필수 여부 | 설명 |
| --- | --- | --- |
| OC | string(필수) | 신청한 API인증값 |
| target | string(필수) | 서비스 대상 (공정거래위원회 : ftc) |
| type | char(필수) | 출력 형태 : HTML/XML/JSON |
| ID | char(필수) | 결정문 일련번호 |

## 5. 출력 결과 (Response Elements) 명세표
| 필드명 | 타입 | 설명 |
| --- | --- | --- |
| 결정문 일련번호 | int | 결정문 일련번호 |
| 문서유형 | string | 출력 형태 : 의결서 / 시정권고서 |
| 사건번호 | string | 사건번호 |
| 사건명 | string | 사건명 |
| 피심정보명 | string | 피심정보명 |
| 피심정보내용 | string | 피심정보내용 |
| 회의종류 | string | 회의종류 |
| 결정번호 | string | 결정번호 |
| 결정일자 | string | 결정일자 |
| 원심결 | string | 원심결 |
| 재산정심결 | string | 재산정심결 |
| 후속심결 | string | 후속심결 |
| 심의정보명 | string | 심의정보명 |
| 심의정보내용 | string | 심의정보내용 |
| 의결문 | string | 의결문 |

| 신청취지 | string | 신청취지 |
| 이유 | string | 이유 |

| 각주내용 | string | 각주내용 |
| 별지 | string | 별지 |
| 결정요지 | string | 결정요지 |

## 6. 에러 코드
- 가이드 페이지에 별도 에러 코드 표는 명시되지 않은 경우가 많습니다. 일반적으로 HTTP 오류, 빈 결과, 또는 XML/JSON 응답의 상태값(제공 시)으로 실패를 판별합니다.

## 7. 비고
- 공정거래위원회에는 시정권고서 영역이 함께 안내됩니다.

## 샘플 요청 및 응답 예시
- 아래 예시는 호출 형태를 빠르게 확인하기 위한 템플릿입니다.
- `{...}` 형태의 식별자 값은 실제 조회 대상 값으로 치환해서 사용합니다.

### 공정거래위원회(ftc) 결정문 본문
#### 요청 예시

```text
https://www.law.go.kr/DRF/lawService.do?OC=test&target=ftc&ID=331&type=HTML
```

```text
https://www.law.go.kr/DRF/lawService.do?OC=test&target=ftc&ID=335&type=XML
```
- 예시 3 (JSON)
```text
http://www.law.go.kr/DRF/lawService.do?OC=test&target=ftc&ID=8111&type=JSON
```
- 예시 4 (HTML)
```text
https://www.law.go.kr/DRF/lawService.do?OC=test&target=acr&ID=53&type=HTML
```
- 예시 5 (XML)
```text
https://www.law.go.kr/DRF/lawService.do?OC=test&target=acr&ID=89&type=XML
```
- 예시 6 (JSON)
```text
https://www.law.go.kr/DRF/lawService.do?OC=test&target=acr&ID=1281&type=JSON
```
- 예시 7 (HTML)

- 예시 9 (JSON)


## 8. 메타
- 정리 기준: 국가법령정보 공동활용 OPEN API 활용가이드 구조화 요약
- 근거 수준: direct
