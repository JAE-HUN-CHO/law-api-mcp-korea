# 조문별 변경 이력 목록 조회

## 1. API 명칭 및 기본 설명
- **API 명칭**: 조문별 변경 이력 목록 조회
- **기본 설명**: 특정 법령 조문별 변경 이력을 조회하는 API

## 2. 가이드 페이지 URL
- https://open.law.go.kr/LSO/openApi/guideResult.do?htmlName=lsJoChgListGuide

## 3. 요청 URL (Endpoint)
- http://www.law.go.kr/DRF/lawService.do?target=lsJoHstInf

## 4. 요청 변수 (Request Parameters) 명세표
| 변수명 | 타입/필수 여부 | 설명 |
| --- | --- | --- |
| OC | string(필수) | 신청한 API인증값 |
| target | string : lsJoHstInf(필수) | 서비스 대상 |
| type | char(필수) | 출력 형태 : XML/JSON |
| ID | char(필수) | 법령 ID |
| JO | int(필수) | 조번호 6자리숫자 : 조번호(4자리)+조가지번호(2자리) (000200 : 2조, 001002 : 10조의 2) |
| display | int | 검색된 결과 개수 (default=20 max=100) |
| page | int | 검색 결과 페이지 (default=1) |

## 5. 출력 결과 (Response Elements) 명세표
| 필드명 | 타입 | 설명 |
| --- | --- | --- |
| 법령ID | int | 법령ID |
| 법령명한글 | int | 법령명(한글) |
| 법령일련번호 | int | 법령일련번호 |
| 공포일자 | int | 공포일자 |
| 공포번호 | int | 공포번호 |
| 제개정구분명 | string | 제개정구분명 |
| 소관부처명 | string | 소관부처명 |
| 소관부처코드 | string | 소관부처코드 |
| 법령구분명 | string | 법령구분명 |
| 시행일자 | int | 시행일자 |
| 조문번호 | int | 조문번호 |
| 변경사유 | int | 변경사유 |
| 조문링크 | int | 변경사유 |
| 조문변경일 | int | 조문변경일 |
| target | string | 검색서비스 대상 |
| totalCnt | int | 검색건수 |

## 6. 에러 코드
- 가이드 페이지에 별도 에러 코드 표는 명시되지 않은 경우가 많습니다. 일반적으로 HTTP 오류, 빈 결과, 또는 XML/JSON 응답의 상태값(제공 시)으로 실패를 판별합니다.

## 샘플 요청 및 응답 예시
- 아래 예시는 호출 형태를 빠르게 확인하기 위한 템플릿입니다.
- `{...}` 형태의 식별자 값은 실제 조회 대상 값으로 치환해서 사용합니다.

#### 요청 예시
- 예시 1 (XML)
```text
http://www.law.go.kr/DRF/lawService.do?OC=test&target=lsJoHstInf&ID=001971&JO=000500&type=XML
```
- 예시 2 (JSON)
```text
http://www.law.go.kr/DRF/lawService.do?OC=test&target=lsJoHstInf&ID=001971&JO=000500&type=JSON
```

#### 응답 예시
- XML 구조 예시
```xml
<response>
  <item>
    <field name="법령ID">12345</field>
    <field name="조문번호">12345</field>
    <field name="조문제목">자동차관리법</field>
    <field name="변경이력목록">예시값</field>
    <field name="조문상세링크">https://www.law.go.kr/...</field>
    <field name="개정일자">20250501</field>
  </item>
</response>
```
- JSON 구조 예시
```json
{
  "data": [
    {
      "법령ID": 12345,
      "조문번호": "12345",
      "조문제목": "자동차관리법",
      "변경이력목록": "예시값",
      "조문상세링크": "https://www.law.go.kr/...",
      "개정일자": 20250501
    }
  ]
}
```

## 8. 메타
- 정리 기준: 국가법령정보 공동활용 OPEN API 활용가이드 구조화 요약
- 근거 수준: direct
