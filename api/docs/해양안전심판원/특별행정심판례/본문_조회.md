# 해양안전심판원 특별행정심판례 본문 조회

## 1. API 명칭 및 기본 설명
- **API 명칭**: 해양안전심판원 특별행정심판례 본문 조회
- **기본 설명**: 해양안전심판원 특별행정심판례 본문 조회 API

## 2. 가이드 페이지 URL
- https://open.law.go.kr/LSO/openApi/guideResult.do?htmlName=specialDeccKmstInfoGuide

## 3. 요청 URL (Endpoint)
- http://www.law.go.kr/DRF/lawService.do?target=kmstSpecialDecc

## 4. 요청 변수 (Request Parameters) 명세표
| 변수명 | 타입/필수 여부 | 설명 |
| --- | --- | --- |
| OC | string, 필수 | 사용자 이메일 ID |
| target | string : kmstSpecialDecc(필수) | 서비스 대상(기관별 special target) |
| type | char, 필수 | 출력 형태 : HTML/XML/JSON |
| ID | char, 필수 | 특별행정심판재결례일련번호 |
| LM | string | 특별행정심판재결례명 |
| fields | string | 응답항목 옵션(HTML 형식에서는 미적용) |

## 5. 출력 결과 (Response Elements) 명세표
| 필드명 | 타입 | 설명 |
| --- | --- | --- |
| 특별행정심판재결례일련번호 | int | 특별행정심판재결례일련번호 |
| 사건명 | string | 사건명 |
| 사건번호 | string | 사건번호 |
| 청구번호 | string | 청구번호 |
| 처분일자 | int | 처분일자 |
| 의결일자 | int | 의결일자 |
| 처분청 | string | 처분청 |
| 재결청 | string | 재결청 |
| 재결례유형명 | string | 재결례유형명 |
| 재결례유형코드 | int | 재결례유형코드 |
| 세목 | string | 세목 |
| 재결요지 | string | 재결요지 |
| 따른결정 | string | 따른결정 |
| 참조결정 | string | 참조결정 |
| 주문 | string | 주문 |
| 청구취지 | string | 청구취지 |
| 이유 | string | 이유 |
| 관련법령 | string | 관련법령 |
| 데이터기준일시 | string | 데이터기준일시 |

## 6. 에러 코드
- 가이드 페이지에 별도 에러 코드 표는 명시되지 않은 경우가 많습니다. 일반적으로 HTTP 오류, 빈 결과, 또는 XML/JSON 응답의 상태값(제공 시)으로 실패를 판별합니다.

## 7. 비고
- 조세심판원 대표 가이드 구조를 기준으로 특별행정심판 계열 API를 정규화했습니다.

## 샘플 요청 및 응답 예시
- 아래 예시는 호출 형태를 빠르게 확인하기 위한 템플릿입니다.
- `{...}` 형태의 식별자 값은 실제 조회 대상 값으로 치환해서 사용합니다.

#### 요청 예시
- 예시 1 (XML)
```text
http://www.law.go.kr/DRF/lawService.do?target=kmstSpecialDecc&OC=test&type=XML&ID={재결례일련번호}
```
- 예시 2 (JSON)
```text
http://www.law.go.kr/DRF/lawService.do?target=kmstSpecialDecc&OC=test&type=JSON&ID={재결례일련번호}
```
