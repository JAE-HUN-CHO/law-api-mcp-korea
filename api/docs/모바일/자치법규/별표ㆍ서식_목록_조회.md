# 모바일 자치법규 별표ㆍ서식 목록 조회

## 1. API 명칭 및 기본 설명
- **API 명칭**: 모바일 자치법규 별표ㆍ서식 목록 조회
- **기본 설명**: 모바일 전용 형식의 자치법규 별표·서식 목록 조회 API

## 2. 가이드 페이지 URL
- https://open.law.go.kr/LSO/openApi/guideResult.do?htmlName=mobOrdinBylListGuide

## 3. 요청 URL (Endpoint)
- http://www.law.go.kr/DRF/lawSearch.do?target=ordinbyl&mobileYn=Y

## 4. 요청 변수 (Request Parameters) 명세표
| 변수명 | 타입/필수 여부 | 설명 |
| --- | --- | --- |
| OC | string(필수) | 신청한 API인증값 |
| target | string : ordinbyl(필수) | 서비스 대상 |
| search | int | 검색범위 (기본 : 1 별표서식명) 2 : 해당자치법규명검색 3 : 별표본문검색 |
| query | string | 법령명에서 검색을 원하는 질의(default=*) |
| display | int | 검색된 결과 개수 (default=20 max=100) |
| page | int | 검색 결과 페이지 (default=1) |
| sort | string | 정렬옵션 (기본 : lasc 별표서식명 오름차순) ldes 별표서식명 내림차순 |
| org | string | 소관부처별 검색(소관부처코드 제공) |
| knd | string | 별표종류 1 : 별표 2 : 서식 3 : 별도 4 : 별지 |
| gana | string | 사전식 검색(ga,na,da…,etc) |
| type | char | 출력 형태 HTML/XML/JSON 생략시 기본값 : XML |
| mobileYn | char : Y (필수) | 모바일여부 |

| 필드명 | 타입 | 설명 |

| section | string | 검색범위 |

| 공포번호 | int | 공포번호 |
| 공포일자 | int | 공포일자 |
| 관련자치법규명 | string | 관련자치법규명 |
| 관련자치법규일련번호 | string | 관련자치법규일련번호 |
| 별표명 | string | 별표명ID |
| 별표번호 | int | 별표번호 |
| 별표서식파일링크 | string | 별표서식파일링크 |
| 별표일련번호 | int | 별표일련번호 |
| 별표자치법규상세링크 | string | 별표자치법규상세링크 |
| 별표종류 | string | 별표종류 |
| 자치법규시행일자 | string | 자치법규시행일자 |
| 전체기관명 | string | 전체기관명 |
| 제개정구분명 | string | 제개정구분명 |
| 지자체기관명 | string | 지자체기관명 |
| 키워드 | string | 검색어 |

## 6. 에러 코드
- 가이드 페이지에 별도 에러 코드 표는 명시되지 않은 경우가 많습니다. 일반적으로 HTTP 오류, 빈 결과, 또는 XML/JSON 응답의 상태값(제공 시)으로 실패를 판별합니다.

## 샘플 요청 및 응답 예시
- 아래 예시는 호출 형태를 빠르게 확인하기 위한 템플릿입니다.

- 예시 1 (XML)
```text
http://www.law.go.kr/DRF/lawSearch.do?OC=test&target=ordinbyl&mobileYn=Y&type=XML
```
- 예시 2 (HTML)
```text
http://www.law.go.kr/DRF/lawSearch.do?OC=test&target=ordinbyl&type=HTML&mobileYn=Y
```
- 예시 3 (JSON)
```text
http://www.law.go.kr/DRF/lawSearch.do?OC=test&target=ordinbyl&type=JSON&mobileYn=Y
```
