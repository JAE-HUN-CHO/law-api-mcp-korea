# 모바일 행정규칙 목록 조회

## 1. API 명칭 및 기본 설명
- **API 명칭**: 모바일 행정규칙 목록 조회
- **기본 설명**: 모바일 전용 형식의 행정규칙 목록 조회 API

## 2. 가이드 페이지 URL
- https://open.law.go.kr/LSO/openApi/guideResult.do?htmlName=mobAdmrulListGuide

## 3. 요청 URL (Endpoint)
- http://www.law.go.kr/DRF/lawSearch.do?target=admrul&mobileYn=Y

## 4. 요청 변수 (Request Parameters) 명세표
| 변수명 | 타입/필수 여부 | 설명 |
| --- | --- | --- |
| OC | string, 필수 | 사용자 이메일 ID |
| target | string : admrul, 필수 | 서비스 대상 |
| type | char, 필수 | 출력 형태 : HTML/XML/JSON 또는 HTML |
| mobileYn | char : Y, 필수 | 모바일 여부 |
| search | int | 검색범위 |
| query | string | 검색어 |
| display | int | 검색 개수 |
| page | int | 결과 페이지 |
| org | string | 소관부처 코드 |
| knd | string | 행정규칙 종류 |
| gana | string | 사전식 검색 |
| sort | string | 정렬옵션 |
| date | int | 발령일자 |

## 5. 출력 결과 (Response Elements) 명세표
| 필드명 | 타입 | 설명 |
| --- | --- | --- |
| target | string | 검색서비스 대상 |
| 키워드 | string | 검색어 |
| section | string | 검색범위 |
| totalCnt | int | 검색건수 |
| page | int | 결과페이지번호 |
| admrul id | int | 결과 번호 |
| 행정규칙일련번호 | int | 행정규칙일련번호 |
| 행정규칙명 | string | 행정규칙명 |
| 행정규칙종류 | string | 행정규칙종류 |
| 발령일자 | int | 발령일자 |
| 발령번호 | int | 발령번호 |
| 소관부처명 | string | 소관부처명 |
| 현행연혁구분 | string | 현행/연혁 구분 |
| 제개정구분코드 | string | 제개정구분코드 |
| 제개정구분명 | string | 제개정구분명 |

| 시행일자 | int | 시행일자 |
| 생성일자 | int | 생성일자 |

## 6. 에러 코드
- 가이드 페이지에 별도 에러 코드 표는 명시되지 않은 경우가 많습니다. 일반적으로 HTTP 오류, 빈 결과, 또는 XML/JSON 응답의 상태값(제공 시)으로 실패를 판별합니다.

## 샘플 요청 및 응답 예시
- 아래 예시는 호출 형태를 빠르게 확인하기 위한 템플릿입니다.

#### 요청 예시
- 예시 1 (XML)
```text
http://www.law.go.kr/DRF/lawSearch.do?target=admrul&mobileYn=Y&OC=test&type=XML&search=1&query=학교&display=20&page=1
```
- 예시 2 (JSON)
```text
http://www.law.go.kr/DRF/lawSearch.do?target=admrul&mobileYn=Y&OC=test&type=JSON&search=1&query=학교&display=20&page=1
```
#### 응답 예시
- XML 구조 예시
```xml
<response>
  <item>
    <field name="target">예시값</field>
    <field name="키워드">학교</field>
    <field name="section">예시값</field>
    <field name="totalCnt">20</field>
    <field name="행정규칙상세링크">https://www.law.go.kr/...</field>
    <field name="page">1</field>
  </item>
</response>
```
- JSON 구조 예시
```json
{
  "data": [
    {
      "target": "예시값",
      "키워드": "학교",
      "section": "예시값",
      "totalCnt": 20,
      "행정규칙상세링크": "https://www.law.go.kr/...",
      "page": 1
    }
  ]
}
```
