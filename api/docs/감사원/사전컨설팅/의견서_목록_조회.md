# 감사원 사전컨설팅 의견서 목록 조회

## 1. API 명칭 및 기본 설명
- **API 명칭**: 감사원 사전컨설팅 의견서 목록 조회
- **기본 설명**: 감사원 사전컨설팅 의견서 목록 조회 API

## 2. 가이드 페이지 URL
- https://open.law.go.kr/LSO/openApi/guideResult.do?htmlName=baiPvcsListGuide

## 3. 요청 URL (Endpoint)
- http://www.law.go.kr/DRF/lawSearch.do?target=baiPvcs

## 4. 요청 변수 (Request Parameters) 명세표
| 변수명 | 타입/필수 여부 | 설명 |
| --- | --- | --- |
| OC | string(필수) | 신청한 API인증값 |
| target | string : baiPvcs(필수) | 서비스 대상 |
| type | char(필수) | 출력 형태 : HTML/XML/JSON |
| search | int | 검색범위 (기본 : 1 의견서명, 2 : 본문검색) |
| query | string | 검색범위에서 검색을 원하는 질의 (정확한 검색을 위한 문자열 검색 query="자동차") |
| display | int | 검색된 결과 개수 (default=20 max=100) |
| page | int | 검색 결과 페이지 (default=1) |
| gana | string | 사전식 검색(ga,na,da…,etc) |
| date | int | 회신일자 |
| rslYd | string | 회신일자 검색(20090101~20090130) |
| sort | string | 정렬옵션 (기본 : lasc 의견서명 오름차순) ldes 의견서명 내림차순 dasc : 회신일자 오름차순 ddes : 회신일자 내림차순 nasc : 접수번호 오름차순 ndes : 접수번호 내림차순 |
| popYn | string | 상세화면 팝업창 여부(팝업창으로 띄우고 싶을 때만 'popYn=Y') |
| fields | string | 응답항목 옵션(의견서명, 접수번호, ...) * 빈 값일 경우 전체 항목 표출 * 출력 형태 HTML일 경우 적용 불가능 |

## 5. 출력 결과 (Response Elements) 명세표
| 필드명 | 타입 | 설명 |
| --- | --- | --- |
| target | string | 검색 대상 |
| 키워드 | string | 키워드 |
| section | string | 검색범위(PvcsNm:의견서명/bdyText:본문) |
| totalCnt | int | 검색결과갯수 |
| page | int | 출력페이지 |
| baiPvcs id | int | 검색결과번호 |
| 감사원사전컨설팅 의견서일련번호 | int | 감사원사전컨설팅의견서일련번호 |
| 의견서명 | string | 의견서명 |
| 회신일자 | string | 회신일자 |
| 신청기관명 | string | 신청기관명 |
| 접수번호 | string | 접수번호 |
| 데이터기준일시 | string | 데이터기준일시 |
| 감사원 사전컨설팅 의견서 상세링크 | string | 감사원 사전컨설팅 의견서상세링크 |

## 6. 에러 코드
- 가이드 페이지에 별도 에러 코드 표는 명시되지 않은 경우가 많습니다. 일반적으로 HTTP 오류, 빈 결과, 또는 XML/JSON 응답의 상태값(제공 시)으로 실패를 판별합니다.

## 샘플 요청 및 응답 예시
- 아래 예시는 호출 형태를 빠르게 확인하기 위한 템플릿입니다.

#### 요청 예시
- 예시 1 (XML)
```text
http://www.law.go.kr/DRF/lawSearch.do?OC=test&target=baiPvcs&type=XML
```
- 예시 2 (HTML)
```text
http://www.law.go.kr/DRF/lawSearch.do?OC=test&target=baiPvcs&type=HTML
```
- 예시 3 (JSON)
```text
http://www.law.go.kr/DRF/lawSearch.do?OC=test&target=baiPvcs&type=JSON
```
- 예시 4 (XML)
```text
http://www.law.go.kr/DRF/lawSearch.do?OC=test&target=baiPvcs&type=XML&gana=ga
```

#### 응답 예시
- XML 구조 예시
```xml
<response>
  <item>
    <field name="target">예시값</field>
    <field name="키워드">계약</field>
    <field name="section">예시값</field>
    <field name="totalCnt">20</field>
    <field name="상세링크">https://www.law.go.kr/...</field>
    <field name="page">1</field>
  </item>
</response>
```
- JSON 구조 예시
```json
{
  "data": [
    {
      "target": "예시값",
      "키워드": "계약",
      "section": "예시값",
      "totalCnt": 20,
      "상세링크": "https://www.law.go.kr/...",
      "page": 1
    }
  ]
}
```
