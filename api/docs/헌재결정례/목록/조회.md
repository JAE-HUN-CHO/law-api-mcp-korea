# 헌재결정례 목록 조회

## 1. API 명칭 및 기본 설명
- **API 명칭**: 헌재결정례 목록 조회
- **기본 설명**: 헌재결정례 목록을 조회하는 API

## 2. 가이드 페이지 URL
- https://open.law.go.kr/LSO/openApi/guideResult.do?htmlName=detcListGuide

## 3. 요청 URL (Endpoint)
- http://www.law.go.kr/DRF/lawSearch.do?target=detc

## 4. 요청 변수 (Request Parameters) 명세표
| 변수명 | 타입/필수 여부 | 설명 |
| --- | --- | --- |
| OC | string, 필수 | 사용자 이메일 ID |
| target | string : detc, 필수 | 서비스 대상(prec/detc/expc/decc 등) |
| type | char, 필수 | 출력 형태 : HTML/XML/JSON |
| search | int | 검색범위(기본: 제목/사건명, 2 본문검색) |
| query | string | 검색어 |
| display | int | 검색된 결과 개수(default=20, max=100) |
| page | int | 검색 결과 페이지(default=1) |
| gana | string | 사전식 검색 |
| sort | string | 정렬옵션 |
| date | int | 선고일/결정일/재결일 |
| popYn | string | 상세화면 팝업 여부 |

## 5. 출력 결과 (Response Elements) 명세표
| 필드명 | 타입 | 설명 |
| --- | --- | --- |
| target | string | 검색 대상 |
| 키워드 | string | 검색어 |
| section | string | 검색범위 |
| totalCnt | int | 검색건수 |
| page | int | 현재 페이지 |
| 헌재결정례일련번호 | int | 헌재결정례일련번호 |
| 사건명 | string | 사건명 |
| 사건번호 | string | 사건번호 |
| 선고일자 | string | 선고일자 |
| 상세링크 | string | 상세링크 |

## 6. 에러 코드
- 가이드 페이지에 별도 에러 코드 표는 명시되지 않은 경우가 많습니다. 일반적으로 HTTP 오류, 빈 결과, 또는 XML/JSON 응답의 상태값(제공 시)으로 실패를 판별합니다.

## 샘플 요청 및 응답 예시
- 아래 예시는 호출 형태를 빠르게 확인하기 위한 템플릿입니다.

#### 요청 예시
- 예시 1 (XML)
```text
http://www.law.go.kr/DRF/lawSearch.do?target=detc&OC=test&type=XML&search=1&query=기본권&display=20&page=1
```
- 예시 2 (JSON)
```text
http://www.law.go.kr/DRF/lawSearch.do?target=detc&OC=test&type=JSON&search=1&query=기본권&display=20&page=1
```
#### 응답 예시
- XML 구조 예시
```xml
<response>
  <item>
    <field name="target">예시값</field>
    <field name="키워드">기본권</field>
    <field name="section">예시값</field>
    <field name="totalCnt">20</field>
    <field name="사건명">예시 사건</field>
    <field name="상세링크">https://www.law.go.kr/...</field>
  </item>
</response>
```
- JSON 구조 예시
```json
{
  "data": [
    {
      "target": "예시값",
      "키워드": "기본권",
      "section": "예시값",
      "totalCnt": 20,
      "사건명": "예시 사건",
      "상세링크": "https://www.law.go.kr/..."
    }
  ]
}
```
