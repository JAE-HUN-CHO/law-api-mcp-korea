# 헌재결정례 본문 조회

## 1. API 명칭 및 기본 설명
- **API 명칭**: 헌재결정례 본문 조회
- **기본 설명**: 헌재결정례 본문을 조회하는 API

## 2. 가이드 페이지 URL
- https://open.law.go.kr/LSO/openApi/guideResult.do?htmlName=detcInfoGuide

## 3. 요청 URL (Endpoint)
- http://www.law.go.kr/DRF/lawService.do?target=detc

## 4. 요청 변수 (Request Parameters) 명세표
| 변수명 | 타입/필수 여부 | 설명 |
| --- | --- | --- |
| OC | string(필수) | 신청한 API인증값 |
| target | string : detc(필수) | 서비스 대상 |
| type | char(필수) | 출력 형태 : HTML/XML/JSON |
| ID | char(필수) | 헌재결정례 일련번호 |
| LM | string | 헌재결정례명 |

## 5. 출력 결과 (Response Elements) 명세표
| 필드명 | 타입 | 설명 |
| --- | --- | --- |
| 헌재결정례일련번호 | int | 헌재결정례일련번호 |
| 종국일자 | int | 종국일자 |
| 사건번호 | string | 사건번호 |
| 사건명 | string | 사건명 |
| 사건종류명 | string | 사건종류명 |
| 사건종류코드 | int | 사건종류코드 |
| 재판부구분코드 | int | 재판부구분코드(전원재판부:430201, 지정재판부:430202) |
| 판시사항 | string | 판시사항 |
| 결정요지 | string | 결정요지 |
| 전문 | string | 전문 |
| 참조조문 | string | 참조조문 |
| 참조판례 | string | 참조판례 |
| 심판대상조문 | string | 심판대상조문 |

## 6. 에러 코드
- 가이드 페이지에 별도 에러 코드 표는 명시되지 않은 경우가 많습니다. 일반적으로 HTTP 오류, 빈 결과, 또는 XML/JSON 응답의 상태값(제공 시)으로 실패를 판별합니다.

## 샘플 요청 및 응답 예시
- 아래 예시는 호출 형태를 빠르게 확인하기 위한 템플릿입니다.
- `{...}` 형태의 식별자 값은 실제 조회 대상 값으로 치환해서 사용합니다.

#### 요청 예시
- 예시 1 (HTML)
```text
http://www.law.go.kr/DRF/lawService.do?OC=test&target=detc&ID=58386&type=HTML
```
- 예시 2 (XML)
```text
http://www.law.go.kr/DRF/lawService.do?OC=test&target=detc&ID=127830&LM=자동차관리법제26조등위헌확인등&type=XML
```
- 예시 3 (JSON)
```text
http://www.law.go.kr/DRF/lawService.do?OC=test&target=detc&ID=58400&type=JSON
```

#### 응답 예시
- XML 구조 예시
```xml
<response>
  <item>
    <field name="헌재결정례일련번호">12345</field>
    <field name="사건명">예시 사건</field>
    <field name="사건번호">12345</field>
    <field name="선고일자">20250501</field>

```
