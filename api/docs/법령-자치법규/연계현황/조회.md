# 법령-자치법규 연계현황 조회

## 1. API 명칭 및 기본 설명
- **API 명칭**: 법령-자치법규 연계현황 조회
- **기본 설명**: 법령과 자치법규의 연계현황을 HTML 화면 형태로 제공하는 API

## 2. 가이드 페이지 URL
- https://open.law.go.kr/LSO/openApi/guideResult.do?htmlName=lsOrdinConGuide

## 3. 요청 URL (Endpoint)
- http://www.law.go.kr/DRF/lawSearch.do?target=drlaw

## 4. 요청 변수 (Request Parameters) 명세표
| 변수명 | 타입/필수 여부 | 설명 |
| --- | --- | --- |
| OC | string, 필수 | 사용자 이메일 ID |
| target | string : drlaw, 필수 | 서비스 대상 |
| type | char, 필수 | 출력 형태 : HTML |

## 5. 출력 결과 (Response Elements) 명세표
| 필드명 | 타입 | 설명 |
| --- | --- | --- |
| 연계현황HTML | string | 법령-자치법규 연계현황 화면 |
| 연계건수 | int | 연계 건수 |
| 관련링크 | string | 관련 링크 |

## 6. 에러 코드
- 가이드 페이지에 별도 에러 코드 표는 명시되지 않은 경우가 많습니다. 일반적으로 HTTP 오류, 빈 결과, 또는 XML/JSON 응답의 상태값(제공 시)으로 실패를 판별합니다.

## 샘플 요청 및 응답 예시
- 아래 예시는 호출 형태를 빠르게 확인하기 위한 템플릿입니다.

- 예시 1 (HTML)
```text
http://www.law.go.kr/DRF/lawSearch.do?target=drlaw&OC=test&type=HTML
```

- HTML 구조 예시

## 8. 메타
- 정리 기준: 국가법령정보 공동활용 OPEN API 활용가이드 구조화 요약
- 근거 수준: direct
