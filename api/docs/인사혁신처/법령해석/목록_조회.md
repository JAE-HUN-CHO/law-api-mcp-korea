# 인사혁신처 법령해석 목록 조회

## 1. API 명칭 및 기본 설명
- **API 명칭**: 인사혁신처 법령해석 목록 조회
- **기본 설명**: 인사혁신처의 중앙부처 1차 법령해석 목록을 조회하는 API

## 2. 가이드 페이지 URL
- https://open.law.go.kr/LSO/openApi/guideResult.do?htmlName=cgmExpcMpmListGuide

## 3. 요청 URL (Endpoint)
- http://www.law.go.kr/DRF/lawSearch.do?target=mpmCgmExpc

## 4. 요청 변수 (Request Parameters) 명세표
| 변수명 | 타입/필수 여부 | 설명 |
| --- | --- | --- |
| OC | string, 필수 | 사용자 이메일 ID(g4c@korea.kr이면 OC=g4c) |
| target | string : mpmCgmExpc(필수) | 서비스 대상(부처별 target 코드) |
| type | char, 필수 | 출력 형태 : HTML/XML/JSON |
| search | int | 검색범위(기본: 1 안건명, 2 본문검색) |
| query | string | 검색어 |
| display | int | 검색된 결과 개수(default=20, max=100) |
| page | int | 검색 결과 페이지(default=1) |
| inq | int | 질의기관코드 |
| rpl | int | 해석기관코드 |
| gana | string | 사전식 검색(ga,na,da…) |
| itmno | int | 안건번호. 지정 시 query는 무시 |
| explYd | string | 해석일자 검색(예: 20090101~20090130) |
| sort | string | 정렬옵션(lasc/ldes/dasc/ddes/nasc/ndes) |
| popYn | string | 상세화면 팝업 여부 |
| fields | string | 응답항목 옵션(HTML 형식에서는 미적용) |

## 5. 출력 결과 (Response Elements) 명세표
| 필드명 | 타입 | 설명 |
| --- | --- | --- |
| target | string | 검색 대상 |
| 키워드 | string | 키워드 |
| section | string | 검색범위(lawNm:안건명, bdyText:본문) |
| totalCnt | int | 검색결과갯수 |
| page | int | 출력페이지 |
| id | int | 검색결과번호 |
| 법령해석일련번호 | int | 법령해석일련번호 |
| 안건명 | string | 안건명 |
| 안건번호 | string | 안건번호 |
| 질의기관코드 | int | 질의기관코드 |
| 질의기관명 | string | 질의기관명 |
| 해석기관코드 | string | 해석기관코드 |
| 해석기관명 | string | 해석기관명 |
| 해석일자 | string | 해석일자 |
| 데이터기준일시 | string | 데이터기준일시 |
| 법령해석상세링크 | string | 법령해석 상세링크 |

## 6. 에러 코드
- 가이드 페이지에 별도 에러 코드 표는 명시되지 않은 경우가 많습니다. 일반적으로 HTTP 오류, 빈 결과, 또는 XML/JSON 응답의 상태값(제공 시)으로 실패를 판별합니다.

## 7. 비고
- 가이드 구조는 중앙부처 1차 해석 API군에서 공통입니다.

## 샘플 요청 및 응답 예시
- 아래 예시는 호출 형태를 빠르게 확인하기 위한 템플릿입니다.

#### 요청 예시
- 예시 1 (XML)
```text
http://www.law.go.kr/DRF/lawSearch.do?target=mpmCgmExpc&OC=test&type=XML&search=1&query=퇴직&display=20&page=1
```
- 예시 2 (JSON)
```text
http://www.law.go.kr/DRF/lawSearch.do?target=mpmCgmExpc&OC=test&type=JSON&search=1&query=퇴직&display=20&page=1
```
#### 응답 예시
- XML 구조 예시
```xml
<response>
  <item>
    <field name="target">예시값</field>
    <field name="키워드">퇴직</field>
    <field name="section">예시값</field>
    <field name="totalCnt">20</field>
    <field name="법령해석상세링크">https://www.law.go.kr/...</field>
    <field name="page">1</field>
  </item>
</response>
```
- JSON 구조 예시
```json
{
  "data": [
    {
      "target": "예시값",
      "키워드": "퇴직",
      "section": "예시값",
      "totalCnt": 20,
      "법령해석상세링크": "https://www.law.go.kr/...",
      "page": 1
    }
  ]
}
```
