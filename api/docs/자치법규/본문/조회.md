# 자치법규 본문 조회

## 1. API 명칭 및 기본 설명
- **API 명칭**: 자치법규 본문 조회
- **기본 설명**: 자치법규 본문을 조회하는 API

## 2. 가이드 페이지 URL
- https://open.law.go.kr/LSO/openApi/guideResult.do?htmlName=ordinInfoGuide

## 3. 요청 URL (Endpoint)
- http://www.law.go.kr/DRF/lawService.do?target=ordin

## 4. 요청 변수 (Request Parameters) 명세표
| 변수명 | 타입/필수 여부 | 설명 |
| --- | --- | --- |
| OC | string(필수) | 신청한 API인증값 |
| type | char(필수) | 출력 형태 HTML/XML/JSON |
| target | string : ordin(필수) | 서비스 대상 |
| ID | char | 자치법규ID |
| MST | string | 자치법규 일련번호 |

## 6. 에러 코드
- 가이드 페이지에 별도 에러 코드 표는 명시되지 않은 경우가 많습니다. 일반적으로 HTTP 오류, 빈 결과, 또는 XML/JSON 응답의 상태값(제공 시)으로 실패를 판별합니다.

## 샘플 요청 및 응답 예시
- 아래 예시는 호출 형태를 빠르게 확인하기 위한 템플릿입니다.
- `{...}` 형태의 식별자 값은 실제 조회 대상 값으로 치환해서 사용합니다.

#### 요청 예시
- 예시 1 (HTML)
```text
http://www.law.go.kr/DRF/lawService.do?OC=test&target=ordin&MST=1316146&type=HTML
```
- 예시 2 (XML)
```text
http://www.law.go.kr/DRF/lawService.do?OC=test&target=ordin&ID=2026666&type=XML
```
- 예시 3 (JSON)
```text
http://www.law.go.kr/DRF/lawService.do?OC=test&target=ordin&ID=2251458&type=JSON
```

#### 응답 예시
- XML 구조 예시
